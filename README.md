Zen Focus is a minimal, modern meditation app designed for Zen practice and concentration.
It provides a calm bilingual interface (English/Russian) with practical tools for daily mindfulness: zazen timer, breath focus mode, beginner guidance, distraction tracking, reflection notes, 
streaks, and weekly progress. The app supports dark/light themes, optional ambient sounds, and offline-friendly behavior.

What the app is for
- Build a consistent meditation habit
- Improve focus by gently returning attention
- Track your practice without complexity
- Reflect after sessions in a private, lightweight way

Main functions
- Daily timer with presets and custom duration
- Breath cue mode (inhale/exhale)
- Beginner mode with short guidance
- Distraction log during active session
- Session history with notes
- Streak and weekly minutes visualization
- RU/EN language switch

It’s built on:
- TypeScript (main coding language)
- React (UI framework)
- Vite (dev server + build tool)
- Capacitor (wraps the web app into Android APK)
- Web APIs (Service Worker + Manifest for PWA/offline, Web Audio for bell/ambient sound)
